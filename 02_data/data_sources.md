# DATA SOURCES

List of sources, API details, update frequency, license:

## price data
- freegoldapi (daily csv + json) — historical backbone
- financialmodelingprep (FMP) — near-realtime quotes
- alphavantage — spot & intraday (free tier limits)
- metalpriceapi (optional paid tier for high-frequency)

## macro
- FRED API — CPI, DXY, 10yr yield, fed dates (daily/weekly)

## news & sentiment
- marketaux / newsapi — aggregated news with metadata
- kitco / reuters RSS — domain-specific feeds
- GDELT — geopolitical event indices (optional)

## etf flows
- GLD/IAU holdings feeds (public ETF data via issuer, FMP)
